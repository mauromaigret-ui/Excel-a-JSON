# Censo 2024 — Excel → JSON (local)

App local en Mac con interfaz web (localhost) para convertir Excel a JSON con auditoría por hoja.

## Documentación
- `docs/01_estructura.md` — estructura de carpetas
- `docs/02_wireframes.md` — wireframes
- `docs/03_especificacion_datos.md` — formato JSON + auditoría
- `docs/04_flujo_y_endpoints.md` — flujo y endpoints

## Arranque local
```bash
./run_local.sh
```

Luego abre `http://localhost:8000` en el navegador.
